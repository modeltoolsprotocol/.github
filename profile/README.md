# Model Tools Protocol

**`--help` for machines.** A minimal standard for making CLI tools LLM-discoverable via `--mtp-describe`.

See the **[protocol spec and rationale](https://github.com/modeltoolsprotocol/modeltoolsprotocol)** for the full story.

## Repositories

| Repository | Install | Description |
|---|---|---|
| [modeltoolsprotocol](https://github.com/modeltoolsprotocol/modeltoolsprotocol) | | Protocol spec, auth spec |
| [mtpcli](https://github.com/modeltoolsprotocol/mtpcli) | `npm i -g @modeltoolsprotocol/mtpcli` | Discover, authenticate, bridge, validate |
| [typescript-sdk](https://github.com/modeltoolsprotocol/typescript-sdk) | `npm i @modeltoolsprotocol/sdk` | Commander.js SDK |
| [python-sdk](https://github.com/modeltoolsprotocol/python-sdk) | `pip install mtp-sdk` | Click SDK |
| [go-sdk](https://github.com/modeltoolsprotocol/go-sdk) | `go get github.com/modeltoolsprotocol/go-sdk` | Cobra SDK |
| [rust-sdk](https://github.com/modeltoolsprotocol/rust-sdk) | `cargo add mtp-sdk` | Clap SDK |
